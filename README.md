# Smart-Blind-Stick
The Smart Blind Stick is one-of-a-kind budget friendly assembled stick installed with all kinds of sensors to ensure efficiency in detecting the obstacles. The stick has been designed in Arduino IDE, and C has been used to develop it. The Stick has been built to run on jagged, irregular terrains. The Stick helps you access the off-road paths, detect stairs, boulders and the crests.
